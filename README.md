# Interactive Applications for Intermediate Microeconomics

A collection of browser-based interactive simulations for use in intermediate microeconomics courses. Each application is a self-contained HTML file — no installation, server, or internet connection required. Students and instructors can open any file directly in a web browser.

Developed by **Steven Matusz**, Professor of Economics  
Department of Economics, Michigan State University

---

## How to Use

**For students:** Click any **▶ Open** link below to run the interactive in your browser. No login or software required.

**For instructors:** Click any **⬇ Download** link to save the `.html` file to your computer. You can then host it on your own LMS (Blackboard, Canvas, D2L, Moodle, etc.), post it to a course website, or distribute it directly to students. Each file is fully self-contained.

## Technical Notes

- All files are **self-contained HTML** with no external dependencies at runtime (Plotly.js and all other libraries load from CDN links embedded in each file)
- Files work in any modern browser: Chrome, Firefox, Safari, Edge
- Files can be used offline once downloaded (CDN libraries will not load without internet)
- All interactives meet **WCAG 2.2 AA accessibility standards** and have been tested with D2L's built-in accessibility checker
