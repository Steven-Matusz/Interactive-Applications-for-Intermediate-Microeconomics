💡 Key Concepts

SRAC curve: Short-run average cost for a given plant size. U-shaped because fixed costs spread over more output but variable costs rise near capacity.

LRAC envelope: The lowest attainable average cost at each output — formed by choosing the optimal plant size. Its shape reveals returns to scale.

Minimum Efficient Scale (MES): The smallest output at which the LRAC reaches its minimum. Industries with large MES tend toward fewer, bigger firms.
